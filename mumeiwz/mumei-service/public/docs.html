简介

沐美服务提供简单易用的REST API，帮助开发者快速集成PDF转换、AI工具、数据处理等功能到应用中。

🚀 基础信息

1

Base URL

https://api.mumei.dev

2

数据格式

所有请求和响应均为 JSON 格式，Content-Type: application/json

3

字符编码

UTF-8

认证

API使用API Token进行认证。在请求头中添加 X-API-Token 字段。

X-API-Token: your_api_token_here

获取API Token

1. 登录用户面板
2. 进入"我的Token"页面
3. 点击"创建Token"
4. 复制生成的Token

// 初始化客户端
const MumeiClient = {
  baseURL: 'https://api.mumei.dev',
  token: 'your_api_token'
};

// PDF转文字
async function convertPDF(file) {
  const formData = new FormData();
  formData.append('pdf', file);
  
  const response = await fetch(`${MumeiClient.baseURL}/api/pdf-to-text`, {
    method: 'POST',
    headers: {
      'X-API-Token': MumeiClient.token
    },
    body: formData
  });
  
  return response.json();
}
